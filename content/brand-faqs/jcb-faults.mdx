Q: How do I pull active JCB fault codes from the dash?
A: Key ON, open Diagnostics / Fault Log on the instrument cluster, note Active vs Stored codes, and confirm with OK. Write down the full code before clearing anything.

Q: Why does the same JCB code come back after a key-cycle reset?
A: That is a hard fault — the ECU still sees the problem. Fix the root cause (filter, sensor, wiring, DEF quality) before expecting the code to stay cleared.

Q: What should I check first for P0087 (fuel rail pressure too low)?
A: Start with the primary fuel filter and lift-pump prime at key-on, then look for air leaks on the suction side. See our P0087 diagnosis guide before replacing expensive rail components.

Q: Can I clear SCR / DEF derate codes by disconnecting the battery?
A: No. Serious emissions and transmission faults live in non-volatile memory. Repair the failed sensor, injector, or wiring, then clear through the dash menu or Service Master as required.

Q: When do I still need a JCB dealer laptop?
A: ECU/TCU replacement programming, injector coding on many Tier 4 machines, and introducing new controllers to the CAN bus still require Service Master. Field parts swaps for filters, NOx sensors, and DEF components usually do not.
